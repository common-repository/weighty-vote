"Vote widget" append votes to your website pages. Plugin works with vote service witch keep vote information from other websites.

== Description ==

Widget placed on the page and allows site visitors to vote. The vote is a choice of one of the proposed options. As long as the user is voted on the page, it increments the counter. During inactive of user voices delegated and can be used the next day. The system of account votes allows interested visitors to stay on your website.

The plugin works through the service, which get and process the votes. One question of voting can be used on several sites that will poll the largest number of people of different interests.

All the colors of the widget can be changed in harmony with the design of the pages.

== Installation ==

To install this plugin, perform the following steps:

1. Upload `vote_widget` folder to the `/wp-content/plugins/` directory;
1. Activate the plugin through the 'Plugins' menu in WordPress;
1. To change the appearance of the widget, click 'Change widget' button in the appended widget in a 'Appearance\Widget' panel;

== Frequently Asked Questions ==

= How I can append a vote in the wordpress sidebar? =

Add vote is very simple, you just need to add a new widget on the panel "Widgets".

= I've added the widget, but the vote did not appear on the page. What to do? =

Widget is not displayed on the page, unless you choose to vote in the widget settings, as well as the absence of connection to the server.

= Where I can find settings page? =

1. Through the 'Plugins' menu in your WordPress admin panel. Find "Vote widget" plugin in the plugins list. Click "Edit" link in appended votes list.
1. Through the 'Appearance' menu 'Widget' item. Append "Vote widget" to sidebar and click "Change widget" button.
1. Through the 'Settings' menu in your admin panel. Select "Vote Widget" subitem. 

= How I can change widget colors? =

In the settings page of widget click on the color edit and select color. After color change click "Save" button for save changes.

= How I can hide a vote? =

In the settings plugin in "Appended votes" list clicking on the link in corresponding column for the widget to change the status "Published" to "Upublished".
